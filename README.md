<H1>Aplicativo em React Native para Prefeitura de Caragutatuba ( Parque Municipal Juqueriquere )</H1>



Produzido nas aulas do IFSP, mentorado pelo proefessor Anderson Roberto Deizeppe  (deizepe.com.br)

Oque é Biodex?

O projeto Biodex é um sistema complexo e completo divido em varias partes

- React Native/Tamagui
- Validação de dados ZOD
- typescript
- API
- Criptografia JWT
